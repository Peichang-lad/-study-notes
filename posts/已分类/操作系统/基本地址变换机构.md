这张图讲的是 **操作系统分页管理中的“基本地址变换机构”**，也就是：

> **CPU产生的逻辑地址，怎么通过页表转换成内存中的物理地址。**

这个是 408 操作系统的重点。

我按做题思路给你拆。

---

## 1. 为什么需要地址变换？

程序运行时看到的是：

```
逻辑地址（虚地址）
```

但是内存真正访问的是：

```
物理地址
```

比如：

程序认为：

```
第10000个字节
```

但它实际可能放在内存：

```
第50000个字节
```

所以需要一个转换机构。

---

## 2. 分页的基本思想

把：

### 逻辑地址空间

切成固定大小：

```
页(page)
```

比如：

```
页大小 = 1KB
```

那么：

```
逻辑地址 = 页号 + 页内偏移
```

写成：

[  
A = P \times L + W  
]

其中：

- A：逻辑地址
    
- P：页号
    
- L：页面大小
    
- W：页内偏移
    

---

举个例：

页大小：

[  
L=1024B  
]

逻辑地址：

[  
A=2050  
]

求：

页号：

[  
P=A/L=2050/1024=2  
]

页内偏移：

[  
W=A%L=2  
]

所以：

```
逻辑地址2050

= 第2页 + 偏移2
```

---

## 3. 页表是什么？

问题来了：

第2页放在哪里？

需要查表。

页表：

|页号|物理块号|
|---|---|
|0|5|
|1|8|
|2|3|
|3|10|

意思：

```
第2页
↓
放在内存第3块
```

---

## 4. 地址转换过程（核心）

假设：

页大小：

[  
L=1KB  
]

逻辑地址：

[  
A=2050  
]

### 第一步：拆地址

得到：

```
页号P=2

页内偏移W=2
```

---

### 第二步：查页表

找到：

```
页号2 → 物理块3
```

所以：

```
块号b=3
```

---

### 第三步：拼物理地址

公式：

[  
E=b\times L+W  
]

代入：

[  
E=3\times1024+2  
]

得到：

[  
E=3074  
]

所以：

最终访问：

```
物理地址3074
```

---

# 5. 图里的 PTR 是什么？

它说：

> 页表寄存器 PTR 保存页表起始地址和长度

意思：

CPU不知道页表在哪。

所以需要一个寄存器：

```
PTR
```

里面放：

```
页表在内存哪里
页表有多长
```

类似：

```
PTR
 |
 ↓
页表
+-----+
|页0 |
|页1 |
|页2 |
+-----+
```

---

# 6. 为什么要比较 P 和 M？

图里第二步：

> 比较页号P和页表长度M

M是什么？

就是：

```
页表有多少项
```

例如：

进程有：

```
4页
```

那么：

```
M=4
```

合法页号：

```
0 1 2 3
```

如果：

```
P=5
```

说明：

访问不存在的页。

产生：

```
越界中断
```

# 7. 手算验证（图上的绿色框）

题目：

页大小：

[  
L=1KB  
]

逻辑地址：

[  
A=2047  
]

求页号：

[  
P=2047/1024=1  
]

页内偏移：

[  
W=2047%1024=1023  
]

所以：

```
页号1
偏移1023
```

---

假设：

页1对应物理块2：

那么：

[  
b=2  
]

物理地址：

[  
E=2\times1024+1023  
]

得到：

[  
E=3071  
]

---

二进制验证也是一样：

因为：

1KB=2¹⁰

所以：

低10位：

```
页内偏移
```

高位：

```
页号
```

拆开再拼回去。

---

# 8. 考试最重要的套路

遇到分页地址转换：

直接四步：

### ① 求页号

[  
P=A/L  
]

### ② 求偏移

[  
W=A%L  
]

### ③ 查页表

找到：

```
页号 → 物理块号b
```

### ④ 拼物理地址

[  
E=bL+W  
]

---

你可以把分页想象成：

```
逻辑空间:

[ 页号 | 页内偏移 ]

        |
        ↓

查快递单(页表)

        |
        ↓

[ 物理块号 | 偏移 ]

        |
        ↓

真正内存地址
```

这块和后面的**快表TLB、两级页表、页面置换**都是一条线上的。你现在学到这里，后面那几个会容易很多。
