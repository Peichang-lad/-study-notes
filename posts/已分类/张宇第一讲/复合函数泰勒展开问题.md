> 对于这种复合函数的极限计算，考研数学中的核心原则是：**先确定外层函数的展开公式（搭骨架），再将内层函数的展开式代入其中（填血肉）。**

对于这种复合函数的极限计算，考研数学中的核心原则是：**先确定外层函数的展开公式（搭骨架），再将内层函数的展开式代入其中（填血肉）。**

具体来说，不要盲目地先把里面展开到很高的阶数，因为你可能并不需要那么多项。最聪明的做法是**先看分母**，确定整个式子需要展开到第几阶（即寻找分母的最低次非零项），然后再去展开分子。

以你图中的第 III 题为例，我们一步步来看如何应用这个策略。

### 1. 确定展开阶数（先看分母）

分母是：

$$\ln(e^{2x} - x^2) - 2x$$

为了使用外层 $\ln(1+u)$ 的标准泰勒公式，我们需要先把内部凑出 $1+u$ 的形式：

$$e^{2x} - x^2 = 1 + 2x + \frac{(2x)^2}{2!} - x^2 + o(x^2) = 1 + 2x + x^2 + o(x^2)$$

此时，令 $u = 2x + x^2$。

接下来套用外层函数 $\ln(1+u) = u - \frac{1}{2}u^2 + o(u^2)$ 的公式：

$$\ln(1 + 2x + x^2) = (2x + x^2) - \frac{1}{2}(2x)^2 + o(x^2) = 2x - x^2 + o(x^2)$$

所以，整个分母展开后为：

$$(2x - x^2) - 2x = -x^2$$

**关键结论：** 分母的最低次非零项是 2 次方。这意味着**分子也只需要展开到 2 次方即可**，多算无益。这就是为什么不能上来就无脑展开内层的原因。

### 2. 按需展开（再算分子）

分子是：

$$\ln(\sin^2 x + e^x) - x$$

同样，先处理内层，凑出 $1+u$ 并且只保留到 2 次方：

$$\sin^2 x \approx x^2$$

$$e^x = 1 + x + \frac{1}{2}x^2 + o(x^2)$$

内层相加：

$$\sin^2 x + e^x = 1 + x + \frac{3}{2}x^2 + o(x^2)$$

此时，令 $v = x + \frac{3}{2}x^2$。

再次套用外层公式 $\ln(1+v) = v - \frac{1}{2}v^2 + o(v^2)$：

$$\ln(1 + x + \frac{3}{2}x^2) = (x + \frac{3}{2}x^2) - \frac{1}{2}(x)^2 + o(x^2) = x + x^2 + o(x^2)$$

所以，整个分子展开后为：

$$(x + x^2) - x = x^2$$

### 3. 得出最终结果

$$\lim_{x \to 0} \frac{x^2}{-x^2} = -1$$

这与你原题旁边手写的答案 `-1` 是一致的。

### 总结：遇到复合函数泰勒展开怎么做？

1. **定阶数：** 永远先算加减法抵消后比较简单的部分（通常是分母），看它展开到第几阶才出现非零项。这决定了你的全局计算量。
    
2. **搭外层：** 明确最外层函数的泰勒公式是什么，比如 $\ln(1+u)$，$\sin(u)$，或者 $e^u$。
    
3. **代内层：** 把内层函数展开到第一步求出的阶数（构成 $u$），然后整体代入外层的公式中。
