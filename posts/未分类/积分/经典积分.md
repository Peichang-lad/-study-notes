# 凑微分：三角函数有理式积分

## 🏷️ 标签
#考研数学 #不定积分 #武忠祥

---

## 💡 核心通法 (待定系数)
对于 $\int \frac{c \sin x + d \cos x}{a \sin x + b \cos x} dx$，令分子：
$$c \sin x + d \cos x = A(a \sin x + b \cos x)' + B(a \sin x + b \cos x)$$
积分结果即为：
$$A \ln|a \sin x + b \cos x| + Bx + C$$

---

## ✍️ 本题推导
**题目：**
$$\int \frac{\sin x}{\sin x + \cos x} dx$$

**解答：**
令 $\sin x = A(\cos x - \sin x) + B(\sin x + \cos x)$
比较系数易得：$A = -\frac{1}{2}, B = \frac{1}{2}$

代入原式：
$$\int \frac{\sin x}{\sin x + \cos x} dx = \int \frac{-\frac{1}{2}(\cos x - \sin x) + \frac{1}{2}(\sin x + \cos x)}{\sin x + \cos x} dx$$
$$= -\frac{1}{2} \int \frac{(\sin x + \cos x)'}{\sin x + \cos x} dx + \frac{1}{2} \int 1 dx$$
$$= -\frac{1}{2} \ln|\sin x + \cos x| + \frac{1}{2}x + C$$
