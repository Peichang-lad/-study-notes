> |              被积函数               | 不定积分结果                                                          

|              被积函数               | 不定积分结果                                                              |
| :-----------------------------: | :------------------------------------------------------------------ |
|    $\frac{1}{a^2 + (bx)^2}$     | $\frac{1}{ab} \arctan\left(\frac{bx}{a}\right) + C$                 |
|    $\frac{1}{a^2 - (bx)^2}$     | $\frac{1}{2ab} \ln\left\vert\frac{a + bx}{a - bx}\right\vert + C$   |
|    $\frac{1}{(bx)^2 - a^2}$     | $\frac{1}{2ab} \ln\left\vert\frac{bx - a}{bx + a}\right\vert + C$   |
| $\frac{1}{\sqrt{a^2 - (bx)^2}}$ | $\frac{1}{b} \arcsin\left(\frac{bx}{a}\right) + C$                  |
| $\frac{1}{\sqrt{(bx)^2 - a^2}}$ | $\frac{1}{b} \ln\left\vert bx + \sqrt{(bx)^2 - a^2}\right\vert + C$ |
| $\frac{1}{\sqrt{(bx)^2 + a^2}}$ | $\frac{1}{b} \ln\left(bx + \sqrt{(bx)^2 + a^2}\right) + C$          |
