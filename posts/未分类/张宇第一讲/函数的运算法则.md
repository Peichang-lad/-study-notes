# 高等数学核心基础速查表

## 第一部分：基本函数与特性

### 1. 五类基本初等函数

|**函数类型**|**通式**|**图像/性质关键词**|
|---|---|---|
|**幂函数**|$y=x^\mu$|抛物线、双曲线、根式；过 $(1,1)$|
|**指数函数**|$y=a^x$|过 $(0,1)$，恒 $>0$，爆炸式增长/衰减|
|**对数函数**|$y=\log_a x$|过 $(1,0)$，定义域 $x>0$，增长缓慢|
|**三角函数**|$y=\sin x, \cos x$|周期震荡，有界|
|**反三角函数**|$y=\arcsin x$|也就是三角函数的反函数|

### 2. 函数的四大特性

|**特性**|**定义/判别**|**图像特征**|
|---|---|---|
|**单调性**|$f'(x)>0 \Rightarrow$ 增<br><br>  <br><br>$f'(x)<0 \Rightarrow$ 减|上升 / 下降|

## 第二部分：代数运算公式

### 1. 指数与对数运算

设定 $a>0, a\ne 1, M>0, N>0$。

|**指数运算 (Indices)**|**对数运算 (Logarithms)**|
|---|---|
|$a^m \cdot a^n = a^{m+n}$|$\log_a (MN) = \log_a M + \log_a N$|
|$\frac{a^m}{a^n} = a^{m-n}$|$\log_a \frac{M}{N} = \log_a M - \log_a N$|
|$(a^m)^n = a^{mn}$|$\log_a M^n = n \log_a M$|
|$a^{-n} = \frac{1}{a^n}$|**换底公式：** $\log_a b = \frac{\ln b}{\ln a}$|
|$a^{\frac{m}{n}} = \sqrt[n]{a^m}$|**恒等式：** $a^{\log_a N} = N$|

## 第三部分：三角函数全家桶

### 1. 和差公式 (展开)

- $\sin(\alpha \pm \beta) = \sin\alpha \cos\beta \pm \cos\alpha \sin\beta$
    
- $\cos(\alpha \pm \beta) = \cos\alpha \cos\beta \mp \sin\alpha \sin\beta$ (**符号相反**)
    
- $\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha \tan\beta}$
    

### 2. 倍角与半角 (降幂)

- **正弦倍角：** $\sin 2\alpha = 2\sin\alpha \cos\alpha$
    
- **余弦倍角：** $\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 2\cos^2\alpha - 1 = 1 - 2\sin^2\alpha$
    
- **降幂公式 (积分必用)：**
    
    - $\sin^2\alpha = \frac{1 - \cos 2\alpha}{2}$
        
    - $\cos^2\alpha = \frac{1 + \cos 2\alpha}{2}$
        

### 3. 辅助角公式 (合成)

$$a\sin x + b\cos x = \sqrt{a^2+b^2} \sin(x + \phi)$$

(其中 $\tan \phi = b/a$)

### 4. 积化和差 (乘积变加减)

- $\sin\alpha \cos\beta = \frac{1}{2} [\sin(\alpha+\beta) + \sin(\alpha-\beta)]$
    
- $\cos\alpha \cos\beta = \frac{1}{2} [\cos(\alpha+\beta) + \cos(\alpha-\beta)]$
    
- $\sin\alpha \sin\beta = -\frac{1}{2} [\cos(\alpha+\beta) - \cos(\alpha-\beta)]$ (**注意负号**)
    

### 5. 和差化积 (加减变乘积)

- $\sin A + \sin B = 2 \sin\frac{A+B}{2} \cos\frac{A-B}{2}$
    
- $\cos A - \cos B = -2 \sin\frac{A+B}{2} \sin\frac{A-B}{2}$ (**注意负号**)
    

### 6. 万能公式

设 $u = \tan\frac{x}{2}$，则：

$\sin x = \frac{2u}{1+u^2}, \quad \cos x = \frac{1-u^2}{1+u^2}, \quad \tan x = \frac{2u}{1-u^2}$
