### 一、 反双曲正弦函数 ($y = \text{arcsinh}(x)$ 或 $\sinh^{-1}(x)$)

反双曲正弦函数是双曲正弦函数 $y = \sinh(x) = \frac{e^x - e^{-x}}{2}$ 的反函数。

由于原函数 $\sinh(x)$ 在整个实数轴上都是单调递增的，因此它的反函数在整个实数轴上都有定义。

#### 1. 数学表达式（对数形式）

反双曲正弦函数可以用自然对数来表示，这是它最重要的解析性质之一：

$$y = \text{arcsinh}(x) = \ln(x + \sqrt{x^2 + 1})$$

#### 2. 基本性质

| **性质**                 | **说明**                                                                                                     |
| ---------------------- | ---------------------------------------------------------------------------------------------------------- |
| **定义域 (Domain)**       | $(-\infty, +\infty)$，即全体实数 $\mathbb{R}$。                                                                   |
| **值域 (Range)**         | $(-\infty, +\infty)$，即全体实数 $\mathbb{R}$。                                                                   |
| **奇偶性 (Parity)**       | **奇函数**。图像关于原点对称。即 $\text{arcsinh}(-x) = -\text{arcsinh}(x)$。                                              |
| **单调性 (Monotonicity)** | 在整个定义域 $(-\infty, +\infty)$ 上**严格单调递增**。                                                                   |
| **截距 (Intercepts)**    | 过原点 $(0, 0)$。当 $x=0$ 时，$y = \ln(0 + \sqrt{1}) = \ln(1) = 0$。                                               |
| **渐近线 (Asymptotes)**   | 无水平或垂直渐近线。但当 $x \to +\infty$ 时，图像表现类似于 $y \approx \ln(2x)$；当 $x \to -\infty$ 时，图像表现类似于 $y \approx \ln(x)$。 |
| **导数 (Derivative)**    | $\frac{d}{dx}(\text{arcsinh}(x)) = \frac{1}{\sqrt{x^2 + 1}}$                                               |
|                        |                                                                                                            |

#### 3. 图像示意

反双曲正弦函数的图像是原函数 $y = \sinh(x)$ 关于直线 $y=x$ 的对称图像。

- 它是一条通过原点的平滑曲线。
    
- 位于第一和第三象限。
    
- 随着 $x$ 的增加，它缓慢上升（对数增长）；随着 $x$ 的减小，它缓慢下降。
    

(图像描述):

想象一条经过原点(0,0)的S形曲线。它在第一象限向右上方延伸，但越来越平缓（类似于对数函数 $\ln(x)$ 的趋势，但起始于原点）；在第三象限向左下方延伸，与第一象限关于原点中心对称。它看起来有点像立方根函数 $y=\sqrt[3]{x}$ 的拉伸版，但增长速率不同。

### 二、 反双曲余弦函数 ($y = \text{arccosh}(x)$ 或 $\cosh^{-1}(x)$)

反双曲余弦函数是双曲余弦函数 $y = \cosh(x) = \frac{e^x + e^{-x}}{2}$ 的反函数。

重要注意： 原函数 $\cosh(x)$ 是一条悬链线（形状像偶次抛物线），它是偶函数，不是一一映射，因此它在整个实数域上没有反函数。

为了定义反函数，我们需要限制原函数的定义域。通常规定取 $\cosh(x)$ 中 $x \ge 0$ 的那一半单调递增分支作为主值分支。

因此，反双曲余弦函数 $y = \text{arccosh}(x)$ 的定义是基于 $\cosh(y) = x$ 且 **$y \ge 0$**。

#### 1. 数学表达式（对数形式）

反双曲余弦函数的对数表达形式为：

$$y = \text{arccosh}(x) = \ln(x + \sqrt{x^2 - 1}) \quad (x \ge 1)$$

#### 2. 基本性质

|**性质**|**说明**|
|---|---|
|**定义域 (Domain)**|$[1, +\infty)$。因为原函数 $\cosh(x)$ 的最小值为 1。|
|**值域 (Range)**|$[0, +\infty)$。这是由于我们限制了原函数的定义域为 $x \ge 0$。|
|**奇偶性 (Parity)**|**非奇非偶函数**。因为定义域关于原点不对称。|
|**单调性 (Monotonicity)**|在定义域 $[1, +\infty)$ 上**严格单调递增**。|
|**截距 (Intercepts)**|与 x 轴交于点 $(1, 0)$。没有 y 轴截距（因为 $x$ 不能为 0）。|
|**渐近线 (Asymptotes)**|无水平或垂直渐近线。当 $x \to +\infty$ 时，图像表现类似于 $y \approx \ln(2x)$。|
|**特殊点**|图像起始于点 $(1, 0)$，此处切线是垂直的（导数无穷大）。|
|**导数 (Derivative)**|$\frac{d}{dx}(\text{arccosh}(x)) = \frac{1}{\sqrt{x^2 - 1}} \quad (x > 1)$|

#### 3. 图像示意

反双曲余弦函数的图像是原函数 $y = \cosh(x)$ (仅右半部分，即 $x \ge 0$ 部分) 关于直线 $y=x$ 的对称图像。

- 图像完全位于第一象限。
    
- 它从 x 轴上的点 $(1, 0)$ 开始。
    
- 在 $(1, 0)$ 点，曲线的切线是垂直的。
    
- 随着 $x$ 增加，曲线向上弯曲并向右延伸，增长速度逐渐变慢（对数增长）。
    

(图像描述):

想象一条起始于点 (1, 0) 的曲线。它看起来像是一个横着放的抛物线的上半部分，但是随着向右延伸，它变得比抛物线更平坦（增长更慢）。它只存在于 $x \ge 1$ 和 $y \ge 0$ 的区域。

### 三、 总结对比简表

| **特征**     | **arcsinh(x)**                                | **arccosh(x)**                                   |
| ---------- | --------------------------------------------- | ------------------------------------------------ |
| **对数形式**   | $\ln(x + \sqrt{x^2 + 1})$                     | $\ln(x + \sqrt{x^2 - 1})$                        |
| **定义域**    | $\mathbb{R}$                                  | $[1, +\infty)$                                   |
| **值域**     | $\mathbb{R}$                                  | $[0, +\infty)$                                   |
| **奇偶性**    | 奇函数                                           | 非奇非偶                                             |
| **起始点/过点** | 过 $(0, 0)$                                    | 起始于 $(1, 0)$                                     |
| **导数特点**   | 分母是 $\sqrt{x^2+1}$ (类似于 $\arctan x$ 导数的分母去根号) | 分母是 $\sqrt{x^2-1}$ (类似于 $\arcsin x$ 导数的分母，但符号相反) |
