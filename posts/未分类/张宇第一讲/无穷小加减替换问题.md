这是考研高数极限计算里一个能真正拉开解题速度差距的技巧。很多教科书为了严谨，直接一刀切地说“加减法绝不能用等价无穷小代换”。但这在实战中如果处处都用泰勒展开，会浪费大量时间。

其实，在满足“不发生抵消”的前提下，加减法是可以安全、快速进行等价代换的。

### 1. 核心法则：什么是“不发生抵消”？

假设当 $x \to 0$ 时，$\alpha \sim \alpha_1$，$\beta \sim \beta_1$。你面对的是一个包含 $\alpha \pm \beta$ 的极限。

**判断标准**：如果在脑海里直接把它们换成等价无穷小后，**$\alpha_1 \pm \beta_1 \neq 0$**（即最高阶的项没有被彻底减没），那么这个加减法代换就是**完全合法且正确**的！

#### 🟢 可以换的情况（没抵消）

**计算**： $\lim_{x \to 0} \frac{\sin x + \tan 2x}{x}$

- **试探代换**：$\sin x \sim x$，$\tan 2x \sim 2x$。
    
- **加减测试**：$x + 2x = 3x \neq 0$。
    
- **结论**：安全！原式 $= \lim_{x \to 0} \frac{3x}{x} = 3$。
    

#### 🔴 绝对不能换的情况（完全抵消）

**计算**： $\lim_{x \to 0} \frac{\tan x - \sin x}{x^3}$

- **试探代换**：$\tan x \sim x$，$\sin x \sim x$。
    
- **加减测试**：$x - x = 0$。
    
- **结论**：发生抵消！如果你硬换，分子变成 $0$，得出极限是 $0$，就彻底掉坑里了。
    

### 2. 为什么抵消了就不能换？（泰勒视角的降维打击）

等价无穷小代换，本质上是**泰勒展开只取了第一项，强行扔掉了后面的高阶无穷小误差**。

- $\sin x = x - \frac{1}{6}x^3 + o(x^3)$
    
- $\tan x = x + \frac{1}{3}x^3 + o(x^3)$
    

当你算 $\tan x - \sin x$ 时，如果只取第一项 $x$，相减变成 $x - x = 0$。但实际上在底层发生的是：

$$ (\tan x - \sin x) = \left(x + \frac{1}{3}x^3\right) - \left(x - \frac{1}{6}x^3\right) = \frac{1}{2}x^3 + o(x^3) $$

发现了吗？一次项确实抵消了，**真正起决定作用的是它们的三次项！** 你用等价代换把一次项减没了，又把人家真正有用的三次项给提前当误差扔了，结果自然是错的。

### 3. 实战神技：化加减为乘除（完美绕过抵消法则）

在考研实战中，遇到像 $\tan x - \sin x$ 这种一看就会抵消的式子，除了苦哈哈地写泰勒展开，更高级的秒杀技巧是：**提取公因式，强行把加减法变成乘除法**。

一旦变成了乘除法，就可以肆无忌惮地使用基础等价代换：

依然是处理 $\tan x - \sin x$：

$$ \tan x - \sin x = \frac{\sin x}{\cos x} - \sin x = \sin x \left(\frac{1}{\cos x} - 1\right) = \sin x \cdot \frac{1 - \cos x}{\cos x} $$

现在式子里全都是乘除法了，开启无脑代换模式（当 $x \to 0$ 时，分母 $\cos x \to 1$ 属于非零常数因子，直接拿走不看）：

- $\sin x \sim x$
    
- $1 - \cos x \sim \frac{1}{2}x^2$
    

直接拼接得出最终的等价式：

$$ \tan x - \sin x \sim x \cdot \frac{1}{2}x^2 = \frac{1}{2}x^3 $$

这个结果和上面用泰勒展开算出来的一模一样，但在考场上，提取公因式的思维速度要比默写泰勒公式并对齐阶数快得多。

### 📌 极限实战口诀总结

1. **加减代换先试探，算完不为零随便换。**
    
2. **一算为零快住手，要么泰勒要么提公因走。**
