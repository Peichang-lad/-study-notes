# 伽马函数 ($\Gamma$)：反常积分降维打击法

> [!abstract] 核心思想
> 遇到积分区间为 $[0, +\infty)$ 且含有指数衰减项（如 $e^{-ax^n}$）的积分，立刻放弃常规求原函数，转而**通过恒等变形强凑 $\Gamma$ 函数定义式**。

## 一、 核心基础知识储备

### 1. 找定义式与约束式
* **定义式**：$$\Gamma(s) = \int_0^{+\infty} x^{s-1} e^{-x} dx$$
* **约束式**：$$s > 0$$ （保证积分收敛）

### 2. 找关系式（三大法宝）
* **递推公式（降阶）**：$$\Gamma(s+1) = s\Gamma(s)$$
* **阶乘扩展（当 n 为正整数时）**：$$\Gamma(n+1) = n!$$
* **黄金特殊值**：$$\Gamma\left(\frac{1}{2}\right) = \sqrt{\pi}$$

---

## 二、 通用解题四步法

> [!important] 实战口诀
> 盯住指数，强令为 $-t$，反解 $x$，套用定义。

1. **找定义式、关系式、约束式**：观察题目是否具备 $[0, +\infty)$ 和 $e^{f(x)}$ 的结构。
2. **做一到两步的逆运算**：
   * 强令指数部分 $f(x) = -t$ （例如令 $t = x^2$ 或 $t = 2x^3$）。
   * 逆运算反解出 $x$。
   * 两边求微分，得出 $dx$。
3. **做恒等变形**：将积分中的 $x$ 和 $dx$ 全部替换为 $t$，把常数项提到积分号外。
4. **联想经典形式**：将剩下的积分部分与 $\Gamma(s) = \int_0^{+\infty} t^{s-1} e^{-t} dt$ 对比，确定 $s$ 的值，最后利用关系式（阶乘或特殊值 $\sqrt{\pi}$）得出结果。

## 三、 实战演练复盘

### 经典题型 1：凑微分直出型
**题目**：计算 $I = \int_0^{+\infty} x^2 e^{-2x^3} dx$

**求解过程**：
1. **逆运算/恒等变形**：令 $t = 2x^3$，两边求导得 $dt = 6x^2 dx$，即 $x^2 dx = \frac{1}{6} dt$。
2. **套用定义**：代入原式得：$$I = \frac{1}{6} \int_0^{+\infty} e^{-t} dt$$
3. **联想经典形式**：这相当于 $\frac{1}{6}\Gamma(1)$。
4. **得出结果**：$\Gamma(1) = 0! = 1$，故结果为 **$1/6$**。

### 经典题型 2：结合特殊值 $\sqrt{\pi}$ 降阶型（极高频）
**题目**：计算 $I = \int_0^{+\infty} x^2 e^{-x^2} dx$

**求解过程**：
1. **逆运算**：令 $t = x^2$，反解得 $x = t^{1/2}$，求导得 $dx = \frac{1}{2}t^{-1/2} dt$。
2. **恒等变形**：代入原式：
   $$I = \int_0^{+\infty} t \cdot e^{-t} \cdot \frac{1}{2}t^{-1/2} dt = \frac{1}{2} \int_0^{+\infty} t^{1/2} e^{-t} dt$$
3. **套用定义**：对比 $\Gamma(s)$ 定义式 $t^{s-1}$，可知 $s-1 = 1/2$，即目标是求 $\frac{1}{2}\Gamma\left(\frac{3}{2}\right)$。
4. **找关系式与特殊值**：
   * 利用递推降阶：$\Gamma\left(\frac{3}{2}\right) = \Gamma\left(\frac{1}{2} + 1\right) = \frac{1}{2}\Gamma\left(\frac{1}{2}\right)$
   * 代入特殊值 $\Gamma\left(\frac{1}{2}\right) = \sqrt{\pi}$
5. **得出结果**：$$I = \frac{1}{2} \times \frac{1}{2}\sqrt{\pi} = \frac{\sqrt{\pi}}{4}$$
