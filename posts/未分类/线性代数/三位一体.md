> 在线代中，矩阵、向量、方程组是三种不同的“语言”。

在线代中，矩阵、向量、方程组是三种不同的“语言”。
设矩阵 $A = (\alpha_1, \alpha_2, \dots, \alpha_n)$，其中 $\alpha_i$ 是列向量。

| **向量语言 (几何/结构)**                                    | **方程组语言 (代数运算)**         | **矩阵语言 (秩与行列式)**                                        |
| --------------------------------------------------- | ------------------------ | ------------------------------------------------------- |
| 向量组 $\alpha_1, \dots, \alpha_n$ **线性无关**            | 齐次方程组 $Ax=0$ **仅有零解**    | $r(A) = n$ (列满秩) / 若是方阵则 $                              |
| 向量组 $\alpha_1, \dots, \alpha_n$ **线性相关**            | 齐次方程组 $Ax=0$ **有非零解**    | $r(A) < n$ / 若是方阵则 $                                    |
| 向量 $\beta$ 能被 $\alpha_1, \dots, \alpha_n$ **线性表示**  | 非齐次方程组 $Ax=\beta$ **有解** | $r(A) = r(A \mid \beta)$                                |
| 向量 $\beta$ **不能**被 $\alpha_1, \dots, \alpha_n$ 线性表示 | 非齐次方程组 $Ax=\beta$ **无解** | $r(A) < r(A \mid \beta)$ 或 $r(A) + 1 = r(A \mid \beta)$ |
