### 一、矩阵等价

**定义：** 设 `A, B` 是同型矩阵，若存在**可逆矩阵 `P, Q`**，使得  
`PAQ = B`  
则称 `A` 与 `B` 等价。

**充要条件：** `A` 与 `B` 等价 ⇔ **秩相等**，即 `r(A) = r(B)`。

**能推出的结论：**
- 秩相同：`r(A) = r(B)`
- 等价标准形相同：都可以通过初等变换化为唯一的形如  
  `[Er  0]`
  `[0   0]` 的矩阵。
- 若 `A, B` 是方阵，行列式是否相等？**不保证**。只有当 `P, Q` 特殊时才有，一般等价不保持行列式。
- **不保持**特征值、特征向量、对称性、正定性等。

> ⚠️ 易混淆点：**矩阵等价** 与 **向量组等价** 不是一回事。  
> 矩阵等价 ⇔ 秩相等；向量组等价 ⇔ 能互相线性表示，向量组等价 ⇒ 矩阵等价，但反之不成立。

---

### 二、矩阵相似

**定义：** 设 `A, B` 为 `n` 阶方阵，若存在**可逆矩阵 `P`**，使得  
`P⁻¹AP = B`  
则称 `A` 与 `B` 相似。

**充要条件：** 很难用简单不变量完全刻画，通常用特征值与 Jordan 标准形。

**能推出的结论（都是必要条件）：**
- 特征多项式相同：`|λE - A| = |λE - B|`
- 特征值完全相同（含重数）
- 行列式相等：`|A| = |B|`
- 迹相等：`tr(A) = tr(B)`
- 秩相等：`r(A) = r(B)`
- 可逆性相同，且逆也相似：`A⁻¹ ∼ B⁻¹`
- 幂也相似：`Aᵏ ∼ Bᵏ`，多项式 `f(A) ∼ f(B)`
- 特征向量的关系：若 `α` 是 `A` 的属于 `λ` 的特征向量，则 `P⁻¹α` 是 `B` 的属于 `λ` 的特征向量。

> ❗ 注意：相似**不保持**对称性。对称矩阵可能相似于非对称矩阵。

---

### 三、矩阵合同

**定义：** 设 `A, B` 为 `n` 阶方阵（考研中几乎专指**实对称矩阵**），若存在**可逆矩阵 `C`**，使得  
`CᵀAC = B`  
则称 `A` 与 `B` 合同。

**充要条件（实对称矩阵）：** `A` 与 `B` 合同 ⇔ **秩相等且正负惯性指数相同**。

**能推出的结论：**
- 秩相同：`r(A) = r(B)`
- 正惯性指数、负惯性指数分别相同
- 对称性保持一致：若 `A` 对称，则 `B` 必对称
- 二次型的规范形唯一
- 若 `A` 可逆，则 `|A|` 与 `|B|`**同号**（由 `|C|²|A| = |B|` 得）
- 正定性、半正定性等保持：若 `A` 正定，则 `B` 正定

> ❗ 合同**不保持**特征值。两个合同矩阵的特征值可以完全不同，只要惯性指数相同即可。

---

### 四、三者关系总结（重点）

```
相似 ⇒ 等价       合同 ⇒ 等价
等价 ⇏ 相似      等价 ⇏ 合同
相似 ⇏ 合同       合同 ⇏ 相似
```

具体展开：

1. **相似 ⇒ 等价，合同 ⇒ 等价**  
   因为 `P⁻¹AP = B` 可取 `P = P, Q = P⁻¹`；`CᵀAC = B` 可取 `P = Cᵀ, Q = C`，都满足等价的定义。等价是三者中最弱的关系，只要求秩相同。

2. **相似与合同一般不互推**  
   - 相似但未必合同：例如 `A = [1 0; 0 2]` 对称，`B = [1 1; 0 2]` 不对称但相似，因不对称，不满足合同前提（考研合同基本限定对称矩阵）。
   - 合同但未必相似：例如 `A = [1 0; 0 1]` 与 `B = [2 0; 0 3]`，惯性指数相同所以合同，但特征值不同，不相似。

3. **实对称矩阵的特殊关系**  
   若 `A, B` 都是**实对称矩阵**：
   - **相似 ⇒ 合同**：因特征值相同，正负惯性指数必然相同，故合同。
   - **合同 ⇏ 相似**：合同只要求惯性指数相同，特征值可随意缩放。
   - **正交相似**（存在正交矩阵 `Q` 使 `Q⁻¹AQ = QᵀAQ = B`）则**同时既是相似又是合同**。

### 五、考研常见命题陷阱速查

| 命题 | 真假 | 说明 |
|------|------|------|
| 相似必等价 | ✅ | 秩相等 |
| 合同必等价 | ✅ | 秩相等 |
| 等价必相似 | ❌ | 秩同特征值不同即可反例 |
| 等价必合同 | ❌ | 等价不要求对称性，合同的前提是对称 |
| 相似必合同 | ❌ | 除非限定实对称矩阵，否则不对称无法谈合同 |
| 合同必相似 | ❌ | 特征值不同即反例 |
| 实对称且相似 ⇒ 合同 | ✅ | 特征值相同，惯性指数必同 |
| 实对称且合同 ⇒ 相似 | ❌ | 惯性指数同特征值可不同 |
| 相似矩阵特征向量相同 | ❌ | 应是用 `P` 转换后的向量 |
| 合同矩阵行列式相等 | ❌ | 只保证同号（可逆时），不保证相等 |
