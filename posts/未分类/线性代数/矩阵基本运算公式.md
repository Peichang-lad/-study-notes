# 矩阵基本运算公式

---

## 一、矩阵加法 / 减法

**条件**：同型矩阵（行数列数相同）

$$A \pm B = (a_{ij} \pm b_{ij})_{m \times n}$$

| 运算律 | 公式 |
|--------|------|
| 交换律 | $A + B = B + A$ |
| 结合律 | $(A + B) + C = A + (B + C)$ |
| 零矩阵 | $A + O = A$ |
| 负矩阵 | $A + (-A) = O$ |

> **记忆**：逐元素加减，规则等同于数的加减。

---

## 二、矩阵数乘

$$kA = (k \cdot a_{ij})_{m \times n}$$

| 运算律 | 公式 |
|--------|------|
| 分配律（对矩阵） | $k(A + B) = kA + kB$ |
| 分配律（对数） | $(k + l)A = kA + lA$ |
| 结合律 | $k(lA) = (kl)A$ |

> **记忆**：数乘每个元素都乘，和分配律完全兼容。

---

## 三、矩阵乘法 ⚠️ 重点

**条件**：$A_{m \times s} \times B_{s \times n} = C_{m \times n}$（A 的列数 = B 的行数）

$$c_{ij} = \sum_{k=1}^{s} a_{ik} \cdot b_{kj}$$

| 运算律 | 结论 |
|--------|------|
| 结合律 | $(AB)C = A(BC)$ ✅ |
| 左分配律 | $A(B + C) = AB + AC$ ✅ |
| 右分配律 | $(A + B)C = AC + BC$ ✅ |
| 数乘结合 | $k(AB) = (kA)B = A(kB)$ ✅ |
| **交换律** | $AB \neq BA$（一般不成立）❌ |

### ⚠️ 核心易错点

| 陷阱 | 说明 |
|------|------|
| $AB = O \nRightarrow A = O \text{ 或 } B = O$ | 非零矩阵乘积可以为零矩阵 |
| $AB = AC \nRightarrow B = C$ | 消去律不成立（除非 A 可逆） |
| $(A+B)^2 \neq A^2 + 2AB + B^2$ | 一般 $AB \neq BA$，不能展开 |
| $(AB)^k \neq A^k B^k$ | 同上 |

> **记忆口诀**：矩阵乘法一切正常，除了**不交换**——这是所有反直觉结论的根源。

---

## 四、矩阵转置

$$A^T = (a_{ji}) \quad \text{即行列互换}$$

| 性质 | 公式 |
|------|------|
| 二次转置 | $(A^T)^T = A$ |
| 加法 | $(A + B)^T = A^T + B^T$ |
| 数乘 | $(kA)^T = kA^T$ |
| **乘法（重点）** | $(AB)^T = \boxed{B^T A^T}$ |
| 幂 | $(A^k)^T = (A^T)^k$ |

> **记忆**：转置穿脱——乘积转置 = **反序转置乘积**，像穿衣服一样：先穿后脱，先脱后穿。
> 口诀：$(AB)^T = B^T A^T$，**"后转置在前"**。

---

## 五、方阵的行列式

| 性质 | 公式 |
|------|------|
| 数乘 | $\|kA\| = k^n \|A\|$（n 为阶数，不是 k\|A\|）|
| 乘积 | $\|AB\| = \|A\| \cdot \|B\|$ |
| 转置 | $\|A^T\| = \|A\|$ |
| 逆 | $\|A^{-1}\| = \frac{1}{\|A\|}$ |
| 幂 | $\|A^k\| = \|A\|^k$ |

> ⚠️ $|A+B| \neq |A| + |B|$，行列式不满足加法分配律。

---

## 六、伴随矩阵（Adjugate）

$$A^* = \text{adj}(A) = (A_{ji})_{n \times n}$$

其中 $A_{ij}$ 是 $a_{ij}$ 的**代数余子式**（注意：下标顺序是 ji，已经做了转置）

### 核心公式

$$A \cdot A^* = A^* \cdot A = |A| \cdot I$$

### 推导链（重点）

| 序号 | 公式 | 说明 |
|------|------|------|
| ① | $A A^* = \|A\| I$ | 伴随定义 |
| ② | $\|A^*\| = \|A\|^{n-1}$ | 对①两边取行列式 |
| ③ | $(kA)^* = k^{n-1} A^*$ | 数乘伴随 |
| ④ | $(A^*)^* = \|A\|^{n-2} A$（$n \geq 3$） | 二次伴随 |
| ⑤ | $(A^*)^{-1} = \frac{A}{\|A\|}$ | 伴随的逆 |
| ⑥ | $(A^{-1})^* = \frac{A}{\|A\|}$ | 逆的伴随 |

### 常见伴随公式

| 公式 | 条件 |
|------|------|
| $(AB)^* = B^* A^*$ | A、B 均可逆 |
| $(A^*)^T = (A^T)^*$ | — |

---

## 七、逆矩阵

$$A^{-1} A = A A^{-1} = I$$

**存在条件**：$|A| \neq 0$（A 可逆 / 非奇异）

### 求逆方法

| 方法 | 公式 |
|------|------|
| **伴随法** | $A^{-1} = \dfrac{A^*}{\|A\|}$ |
| **初等变换法** | $(A \mid I) \xrightarrow{\text{行变换}} (I \mid A^{-1})$ |

### 运算性质

| 性质 | 公式 |
|------|------|
| 二次逆 | $(A^{-1})^{-1} = A$ |
| 数乘逆 | $(kA)^{-1} = \frac{1}{k} A^{-1}$ |
| **乘积逆（重点）** | $(AB)^{-1} = \boxed{B^{-1} A^{-1}}$ |
| 转置逆 | $(A^T)^{-1} = (A^{-1})^T$ |
| 行列式 | $\|A^{-1}\| = \frac{1}{\|A\|}$ |

> **记忆**：逆和转置规律一致——**反序**。$(AB)^{-1} = B^{-1}A^{-1}$，口诀："先逆者在后"。

---

## 八、转置 · 伴随 · 逆 三方对比总表

| 操作 | 乘积 | 数乘 | 自身再操作 |
|------|------|------|-----------|
| **转置** $A^T$ | $(AB)^T = \color{red}{B^T A^T}$ | $(kA)^T = kA^T$ | $(A^T)^T = A$ |
| **伴随** $A^*$ | $(AB)^* = \color{red}{B^* A^*}$ | $(kA)^* = k^{n-1}A^*$ | $(A^*)^* = \|A\|^{n-2}A$ |
| **逆** $A^{-1}$ | $(AB)^{-1} = \color{red}{B^{-1} A^{-1}}$ | $(kA)^{-1} = \frac{1}{k}A^{-1}$ | $(A^{-1})^{-1} = A$ |

> **统一规律**：三者乘积都反序！**转置、伴随、逆，乘积全反序。**
>
> 数乘区别：转置不变 $k$，逆变 $\frac{1}{k}$，伴随变 $k^{n-1}$。

---

## 九、四者关系链

```
        |A|I = A A* = A* A
              │
              ▼
         A* = |A| · A⁻¹          A⁻¹ = A* / |A|
              │                        │
              ▼                        ▼
        |A*| = |A|ⁿ⁻¹           |A⁻¹| = 1/|A|
```

> **记忆核心**：掌握 $A A^* = |A| I$ 这一条，伴随和逆的所有公式都能推出来。

## 十、特殊矩阵速记

| 矩阵类型 | 定义 | 关键性质 |
|----------|------|----------|
| **对称矩阵** | $A^T = A$ | 元素以主对角线对称 |
| **反对称矩阵** | $A^T = -A$ | 主对角线全为 0 |
| **正交矩阵** | $A^T A = A A^T = I$ | $A^{-1} = A^T$ |
| **对角矩阵** | 非对角元为 0 | 乘积可交换 |
| **单位矩阵** | 对角全 1 | $AI = IA = A$ |

> **正交矩阵特别记忆**：逆 = 转置，是唯一两者相等的矩阵类型。
